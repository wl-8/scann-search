## 项目概述

这是一个面向单细胞高维向量数据的 ANN 检索系统：
- **后端**：FastAPI + SQLAlchemy + FAISS
- **前端**：Vue 3 + Vite + TypeScript + Ant Design Vue

---

## 后端启动

### 方式一：使用 Conda（推荐，避免二进制包编译问题）

```bash
# 1. 创建并激活环境
conda env create -f environment.yml
conda activate scann-search

# 2. 启动服务
cd backend
uvicorn app.main:app --reload
```

### 方式二：纯 pip 安装

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

> ⚠️ 注意：`requirements.txt` 提到二进制包（numpy/scipy/faiss-cpu/hnswlib/scanpy/anndata）建议用 conda 安装，避免 Windows 下 pip 编译 C++ 失败。

---

## 前端启动

```bash
cd frontend
npm install        # 首次运行需要安装依赖
npm run dev
```

启动后访问 `http://localhost:5179/`

## 开发模式 Mock

前端内置了 mock API，后端不可用时也能演示完整流程。搜索结果会自动回退到 mock 数据。

如果需要连接真实后端，在 `frontend` 目录创建 `.env` 文件：

```env
VITE_API_BASE=http://localhost:8000/api
```

## 索引构建参数说明

### 支持的算法

| 算法 | 说明 |
|------|------|
| `flat` | 暴力精确搜索（baseline，无参数） |
| `hnsw` | 分层可导航小世界图（推荐） |
| `ivf` | 倒排索引 + 量化 |
| `lsh` | 局部敏感哈希 |
| `pq` | 产品量化 |
| `opq` | 优化产品量化 |
| `ivf_pq` | IVF + PQ 混合 |
| `ivf_hnsw` | IVF + HNSW 混合 |

### 参数配置（JSON 格式）

#### HNSW（推荐，默认参数）

```json
{}
```

或自定义参数：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `M` | 16 | 每个节点的连接数，越大越精确但越慢 |
| `ef_construction` | 200 | 构建时的搜索范围，越大越精确但越慢 |
| `ef_search` | 50 | 搜索时的搜索范围，越大越精确但越慢 |
| `space` | `l2` | 距离度量：`l2`（欧氏）/ `ip`（内积）/ `cosine`（余弦）|

#### IVF

```json
{
  "nlist": 100,
  "nprobe": 10
}
```

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `nlist` | 100 | 聚类中心数 |
| `nprobe` | 10 | 搜索时探查的聚类数 |

#### Flat（无需参数）

```json
{}
```
